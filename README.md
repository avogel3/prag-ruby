# prag-ruby

Ruby Programming Tutorial by the Pragmatic Studio.

* The follow along programming is the **ruby\_studio** directory. 
* The diy exercises live in the **studio\_game** directory.


### Usage

Run the tests with:

```bash
rspec the_spec.rb
```

Run files with: 

```bash
ruby filename.rb
```
